# UOW New Tab — Session Dates

A lightweight Chrome extension that replaces your new tab page with the current date, time, and all active UOW session weeks.

## What it shows

Every time you open a new tab, you'll see:
- Live clock and today's date
- All currently active UOW sessions and what week they're in
- Named periods like Mid-Session Recess, Study Recess, Final Exams — shown in place of a week number when relevant

Sessions covered: Autumn 2026, Spring 2026, Annual 2026, Summer 2026/27, Trimester 1–3 2026.

---

## Install (Chrome — Developer Mode)

> No Chrome Web Store account needed. Works on Mac and Windows.

1. **Download this repo**
   - Click the green **Code** button → **Download ZIP**
   - Unzip the folder somewhere you'll remember (e.g. Documents)

2. **Open Chrome extensions**
   - Go to `chrome://extensions` in your browser

3. **Enable Developer Mode**
   - Toggle the switch in the top-right corner

4. **Load the extension**
   - Click **Load unpacked**
   - Select the unzipped `uow-new-tab` folder

5. **Open a new tab** — you're done ✅

---

## Updates

When the extension is updated (e.g. new session dates added):

1. Download the latest version from GitHub (or `git pull` if you're using Git)
2. Go to `chrome://extensions`
3. Click the **refresh icon** on the UOW Session Dates card
